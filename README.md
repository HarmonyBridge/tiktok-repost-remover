## About the Project

Simple, privacy-focused tool to **remove all reposted videos** from your TikTok profile.  
TikTok doesn't offer native bulk delete for reposts — this tool automates the process directly in your browser session (no API keys, no passwords stored).

Great for:
- Cleaning old/spammy reposts
- Refreshing profile content
- Removing accidental mass reposts
- Preparing account for new niche/content direction

Works locally in your browser — no account data leaves your device.

## ⚠️ Important Warning

**Automating actions on TikTok may violate their Terms of Service.**  
Possible risks:
- Temporary action blocks
- Shadowban or account restrictions
- In rare cases — permanent ban (especially if run too fast/aggressively)

**Use at your own risk, preferably with pacing/delay enabled.**  
Author is **not responsible** for any account issues, restrictions, or lost visibility.  
Recommended: run on alt/test accounts first, use slow speed (1–3 sec delay).

2. Extract the archive (**Extract All**)
3. Run **Launcher.exe** (or Launcher.bat / start.bat — depending on version)
4. Follow the launcher instructions

Everything bundled — no Python install, no extra extensions needed.
